# MCU竞品数据库 — 数据字典与口径声明

**维护人**：ST数字营销/数据组  
**最后更新**：2026-05-19  
**货币**：人民币（CNY），M$换算见 `fx_rates.json`  
**版本**：v2.0（基于2023-2025年报及季报实证验证）

---

## 一、数据置信度定义

| 级别 | 含义 | 使用场景 |
|------|------|---------|
| `high` | 年报分产品表直接披露，可直接引用 | 对外报告、领导汇报 |
| `medium` | 年报分类含非MCU成分，或需系数换算，来源可追溯 | 内部分析、趋势判断 |
| `low` | 管理层IR口头估算或系数推算，无年报直接来源 | 仅供参考，不建议单独引用 |
| `forecast` | 分析师一致预期或外推，尚未发布年报 | 趋势方向参考 |

---

## 二、data_type 字段定义

| 值 | 含义 |
|----|------|
| `actual` | 已发布年报/季报数据，经审计 |
| `forecast` | 分析师一致预期，未经审计 |
| `estimated` | 内部系数估算，需附 `note` 说明方法 |

---

## 三、汽车MCU状态定义（auto_mcu_status）

| 值 | 含义 |
|----|------|
| `volume` | 规模量产，有可统计收入 |
| `ramping` | 批量出货爬坡，收入体量小 |
| `pilot` | 小批量试产 / AEC-Q100认证阶段 |
| `rd_only` | 仅研发投入，无产品收入 |
| `na` | 不涉及汽车MCU |

---

## 六、净利润估算方法（如需使用）

年报只提供收入和毛利，MCU产品线净利润需估算：

```
估算MCU净利润 = MCU营业收入 × (MCU毛利率 - 公司整体期间费用率)
期间费用率 = (销售费用 + 管理费用 + 研发费用 + 财务费用) / 公司总营业收入
```

**局限性**：
- 研发密集型公司（兆易、复旦微）的MCU研发费用占比通常高于其收入占比，此公式低估费用，净利润偏乐观
- 结果为数量级参考，不建议正式引用，引用时须附局限性说明
